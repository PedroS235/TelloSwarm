- Some notions to know when running the simulation             -
================================================================
 -> Left-click adds a start at the mouse cursor position.
 -> Right-click adds a goal at the mouse cursor position.
 -> <Return> starts the simulation.
 -> <Esc> clears the simulation.
 -> Logs are written to standard output.
